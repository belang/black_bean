= 可计算性 =
一个计算由两部分构成：当前动作（动作）和下一动作（指示）。

对于处理器，每个动作是一条机器指令，而每个指示也是一条机器指令。

一个计算是指令的组合。一个任务是计算的组合。

使用每拍单动作的方式，更加灵活。复杂的指令，可以由动作指令编码而成。

= 计算控制 =
计算顺序的控制核心是控制器。
控制器读取指令、判断是否要译码、控制下一指令提取。
